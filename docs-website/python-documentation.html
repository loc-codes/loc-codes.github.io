Introduction

Python is a high-level, interpreted programming language that is widely used for developing applications in various fields such as:

  • Web Development
  • Scientific Computing
  • Data Analysis
  • Artificial Intelligence
Python was first released in 1991 by Guido van Rossum and has since then become one of the most popular programming languages in the world.

Python is designed to be easy to read and write, with a clean syntax and intuitive object-oriented approach. It has a vast standard library that provides many useful modules and functions for various tasks such as file I/O, networking, regular expressions, and more. Additionally, Python has a large community of developers who have created many third-party packages that extend its capabilities even further.

Hello, world!

Printing "Hello, World!" is often the first program that developers learn to write in a new programming language.

To print "Hello, World!" in Python, you can use the built-in print() function. This function takes one or more arguments and prints them to the console.

Here is an example of how to use the print() function to print "Hello, World!" in Python:

print("Hello, world!")

When you run this program in IDLE, you will see the output "Hello, World!" printed to the console.

>>> print('Hello, world!')
... Hello, world!

Congratulations! You have just written your first Python program!

Variables

In programming, variables are used to store data that can be used later in the program. In Python, variables are created when you assign a value to them using the assignment operator =.

Here is an example of how to create a variable and assign it a value in Python:

message = "Hello, World!"

In this case, we are creating a variable called message and assigning it the value "Hello, World!".

Python variables can hold many different types of data. You do not need to declare the type of a variable in Python. Python automatically works this out!

Variables in Python can also be reassigned with a new value at any time. For example:

>>> x = 5
>>> print(x)
... 5

>>> x = "Hello"
>>> print(x)
... Hello

In this example, we first assign the integer value 5 to the variable x, and then we reassign it with the string "Hello".

In addition to storing data, variables can also be used in expressions and calculations. For example:

>>> x = 5
>>> y = 3
>>> z = x + y
>>> print(z)
... 8

In this case, we are creating two variables x and y, assigning them the values 5 and 3 respectively, and then creating a third variable z which is the sum of x and y.

Strings, Integers and Floats

In Python, three of the most commonly used data types are:

  • Strings
  • Integers
  • Floats
Let's take a closer look at each of these types and how to work with them in Python.

Strings

A string is a sequence of characters, enclosed in either single or double quotes.

Here is an example of a string in Python:

>>> name = "John"

You can concatenate two or more strings using the + operator:

>>> greeting = "Hello"
>>> name = "John"
>>> message = greeting + " " + name

>>> print(message)
... Hello John

Integers

An integer is a whole number, either positive or negative. You can perform operations on integers using the standard mathematical operators such as +, -, *, and /.

Recall the example from the last section:

>>> x = 5
>>> y = 3
>>> z = x + y
>>> print(z)
... 8

Floats

A float is a number with a decimal point. You can perform operations on floats in the same way as integers.

Here is an example:

>>> x = 3.14
>>> y = 2.0
>>> z = x * y
>>> print(z)
... 6.28

Boolean Logic

Boolean logic is a fundamental concept in programming that deals with true/false values. In Python, the two Boolean values are True and False

You can create Boolean expressions using comparison operators such as ==, !=, >, <, >=, and <=.

Here are some examples:

>>> x = 5
>>> y = 3

>>> print(x > y)
... True

>>> print(x = y)
... False

Boolean expressions can also be combined using logical operators such as "and", "or", and "not".

Here are some examples:

>>> x = 5
>>> y = 3
>>> z = 7

>>> print(x > y and x < z)
... True

>>> print(x < y or x < z)
... True

>>> print(not x > y)
... False

In this case, we are using the and, or, and not operators to create more complex Boolean expressions.
